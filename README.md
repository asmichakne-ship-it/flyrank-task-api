# Task API

A simple CRUD REST API built with Node.js and Express.

## Features

- GET all tasks
- GET task by ID
- POST new task
- PUT update task
- DELETE task
- Swagger API documentation

## Technologies

- Node.js
- Express
- Swagger UI
- JavaScript

## Installation

```bash
npm install
```

## Run

```bash
node app.js
```

Open:

```
http://localhost:3000/docs
```

to view the Swagger documentation.
